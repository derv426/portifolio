Quem sou eu?

Olá! Meu nome é João Pedro e sou um desenvolvedor web em formação, com o sonho de me tornar um desenvolvedor full-stack. Atualmente, tenho conhecimento em HTML e CSS e estou me aprofundando em JavaScript para criar aplicações mais dinâmicas e interativas. Tenho interesse em aprender TypeScript para escrever códigos mais organizados e seguros, além de estudar React, uma das bibliotecas mais populares para o desenvolvimento front-end. Também quero explorar Tailwind CSS para facilitar a estilização de interfaces modernas e responsivas. No futuro, pretendo expandir meus conhecimentos para o back-end, estudando Node.js, Express e bancos de dados como MongoDB e PostgreSQL. Meu objetivo é dominar tanto o front-end quanto o back-end para criar aplicações completas e eficientes. Sou apaixonado por tecnologia e estou sempre buscando aprender mais para me tornar um desenvolvedor cada vez melhor.
